# Simple Chat

A full-stack, real-time chat application built as a **sample project** to showcase coding style, file structure, and design system knowledge. This project demonstrates my understanding of frontend and backend development, realtime communication, and problem-solving with modern web technologies.

---

## Features

- **Real-time messaging** with Socket.IO
- **Frontend**: React, Redux, Ant Design, Socket.IO client
- **Backend**: Express, Socket.IO
- Clean file structure and modular code for easy scalability
- Designed as a learning reference for developers exploring realtime apps

> ⚠️ This is a **sample project**. Some features are intentionally missing to focus on showcasing structure and coding patterns.

---

## Tech Stack

**Frontend:**
- React
- Redux
- Socket.IO client
- Ant Design

**Backend:**
- Node.js
- Express
- Socket.IO
